pairipcore.insn.O(name: str, opcode: int, format_id: str) → int[source]#

Registers an opcode with its corresponding format and name.

This function associates an opcode with a specific instruction format (represented by the format ID) and a mnemonic name. The format and name are stored in the FormatIDs and OpcodeIDs dictionaries, respectively.

Args:

name (str): The mnemonic name of the opcode. If the name is "_", it will be auto-generated based on the format ID. opcode (int): The numeric opcode to be registered. format_id (str): The format ID string that describes the structure of the instruction.

Returns:

int: The opcode that was registered.

Parameters:

  • name (str)

  • opcode (int)

  • format_id (str)

int
